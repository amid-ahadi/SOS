معرفی سیستم

این بخش یک نمای کلی از سیستم مدیریت دفتر وکلا، هدف اصلی و تاریخچه آن ارائه می‌دهد. این نرم‌افزار برای ساده‌سازی و بهینه‌سازی فرآیندهای کاری در دفاتر حقوقی طراحی شده است.

سیستم مدیریت دفتر وکلا (Advocate Office Management System) یک راهکار جامع برای مدیریت عملیات روزمره دفاتر حقوقی و وکالت است. این سیستم، که در ابتدا از Envato خریداری شده بود، ابزارهای لازم برای سازماندهی پرونده‌ها، مشتریان، جلسات، حسابداری و ارتباطات را فراهم می‌کند. هدف این سیستم، بهبود بهره‌وری و کارایی در دفاتر وکالت از طریق یک پلتفرم متمرکز و کاربرپسند است.

System Overview

This section provides an overview of the Advocate Office Management System, its primary purpose, and its history. The software is designed to simplify and optimize workflows in legal firms.

Advocate Office Management System is a comprehensive solution designed to streamline the daily operations of legal firms and advocate offices. Originally acquired from Envato, this system provides essential tools for organizing cases, clients, appointments, accounting, and communications. Its primary goal is to enhance productivity and efficiency in legal practices through a centralized and user-friendly platform.

قابلیت‌های اصلی

در این بخش، مهم‌ترین قابلیت‌هایی که این سیستم برای مدیریت دفاتر وکالت ارائه می‌دهد، به صورت بصری نمایش داده شده است. هر قابلیت برای رفع یکی از نیازهای کلیدی دفاتر حقوقی طراحی شده است.

نیازمندی‌های سرور

برای نصب و اجرای موفقیت‌آمیز سیستم، سرور شما باید این پیش‌نیازهای فنی را داشته باشد. مطمئن شوید که همه موارد قبل از شروع نصب فراهم هستند.

Server Requirements

For a successful installation and operation, your server must meet these technical prerequisites. Ensure all items are checked before starting.

  • 🖥️ سرور محلی: XAMPP, WAMP, AMPPS یا هر وب سرور مشابه.Local Server: XAMPP, WAMP, AMPPS, or any similar web server.
  • 🐘 PHP: نسخه ۷.۴ به بالا (نسخه ۸.۱ بهینه است).PHP: Version 7.4 or higher (version 8.1 is optimized).
  • 🐬 MySQL: نسخه ۵.۷ به بالا.MySQL: Version 5.7 or higher.
  • 🔄 Mod Rewrite: ماژول `mod_rewrite` آپاچی باید فعال باشد.Mod Rewrite: Apache's `mod_rewrite` module must be enabled.

راهنمای نصب گام به گام

این راهنما شما را در تمام مراحل نصب سیستم، از آپلود فایل‌ها تا ورود اولیه، همراهی می‌کند. برای مشاهده جزئیات هر مرحله، روی عنوان آن کلیک کنید. نمودار زیر مراحل اصلی نصب را نشان می‌دهد.

Step-by-Step Installation Guide

This guide will walk you through the entire installation process, from uploading files to the initial login. Click on each step to view its details. The chart below illustrates the main installation phases.

مرحله ۱: آپلود فایل‌هاStep 1: Upload Files ▼

فایل‌های پروژه را از طریق FTP یا پنل هاست خود به روت وب‌سایت (عموماً پوشه `public_html` یا `htdocs`) منتقل کنید.Transfer the project files via FTP or your hosting panel to the website root (usually the `public_html` or `htdocs` folder).

مرحله ۲: تنظیم دیتابیسStep 2: Database Setup ▼

یک دیتابیس جدید در phpMyAdmin ایجاد کرده و فایل `database.sql` را در آن Import کنید.Create a new database in phpMyAdmin and import the `database.sql` file into it.

مرحله ۳: نصب وبیStep 3: Web Installer ▼

با مراجعه به آدرس وب‌سایت خود، مراحل نصب‌کننده وبی را دنبال کرده و اطلاعات دیتابیس و مدیر را وارد کنید.Navigate to your website's address, follow the web installer steps, and enter the database and administrator credentials.

مرحله ۴: ورود به سیستمStep 4: Login ▼

پس از اتمام نصب، با نام کاربری و رمز عبوری که ایجاد کرده‌اید، وارد پنل مدیریت شوید.After installation is complete, log in to the admin panel with the username and password you created.
